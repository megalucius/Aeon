# Aeon
A Single player RPG in development
Game Controls

WASD To move
space to jump
mouse to move camera
J - move 3
K - move 2
Left click - move 1
1 - teleport to area
2 - deploy unit 
3 - quick select weapon (uses placeholder image)
4 - pause menu 

The game can also be played with a PS4 Remote with Inputmapper as well
